Zorg ervoor dat Python 3.11 of hoger geïnstalleerd is.

Installeer (via "pip install" in CMD) de volgende libraries:
	pygame
	asyncio

Navigeer naar de map waar "main.py" zich bevindt.

Open hier een CMD en start het spel met "python main.py"




Wil je zelf de web-versie bouwen, zorg dan dat je de volgende library geïnstalleerd hebt:
	pygbag

Open vervolgens een CMD in de map waarin zich "main.py" bevindt en run het volgende commando (zonder aanhalingstekens):

"pygbag --title Bibber --app_name Bibber --ume_block 0 --can_close 0 --build main.py"
